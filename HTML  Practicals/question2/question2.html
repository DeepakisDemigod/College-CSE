These are Types of Lists

1. Ordered List
The ol tag defines an ordered list. An ordered list can be numerical or alphabetical. The li tag is used to define each list item.

for example :
    Fruits
  1. Apple
  2. Mango
  3. Lichi
2. Unordered List
An unordered list starts with the ul tag. Each list item starts with the li tag. The list items will be marked with bullets (small black circles) by default. for example :
    Fruits
  • Apple
  • Mango
  • Lichi
